# Radhakrishna Cinemax API

Standalone backend branch for the Radhakrishna Cinemax API.

## Scripts

```bash
npm install
npm run dev
npm start
```

## Render

Use this branch as a backend web service.

- Build command: `npm ci --legacy-peer-deps`
- Start command: `NODE_ENV=production npm start`
- Health check: `/api/health`
